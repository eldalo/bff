# BFF

## Description

BFF

- based on the [NestJS](https://nestjs.com/) framework.
- with Graphql

## Environment Variables

| Variable Name            | Default Value           | Required |
| ------------------------ | ----------------------- | -------- |
| `APEX_CLIENT_API_URL`    | `http://localhost:3000` | Yes      |
| `APEX_PORTFOLIO_API_URL` | `http://localhost:3000` | Yes      |

## Project setup

```bash
$ npm install
```

## Compile typings

```bash
$ npm run generate:typings
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Open browser

```bash
http://localhost:3001/graphql
```
